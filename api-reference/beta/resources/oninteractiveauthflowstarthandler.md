# onInteractiveAuthFlowStartHandler resource type

Namespace: microsoft.graph

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Base/abstract class for configuration for what to invoke if the onInteractiveAuthFlowStart event resolves to this listener. Each handler available for this event type is a subtype of this abstract complex type.
